### Personal Finances Analysis

A solution to extract data from OFX file and transform in insights using Generative AI.

The main objective is to simplify personal finance analysis through a pipeline that consolidate data in one place and using Generative AI to classify transactions and analyse data to give you insights.


**Features**
- Dataprep pipeline to clean and create new features
- Using LLM to classify transactions
- Using Google Sheet to storage new transactions
