### Personal Finances Analysis

A solution to extract data from OFX files and transform in insights using Generative AI.

The main objective is to simplify personal finance analysis through a pipeline that consolidate data in one place, using Generative AI to classify out transactions and analyse data to give you insights.

**Features**
- Dataprep pipeline to clean and create new data features
- Using LLM to classify credit card transactions
- Streamlit dashboard to analyse data and insights


## How to Activate the Dev Env and Run the Application
```
python -m venv .venv

.venv\Scripts\activate

pip install -r requirements.txt
```
